git
1. download -> clone.
    (clone은 처음 다운로드 받을 때만 사용.)

2. 새로 생긴 파일 혹은 변경된(수정 or 삭제) 파일을 git 에 추가 해준다. (add)

3. 추가한 내용을 git에 저장한다.(commit)

4. git에 저장한 내용을 업로드(git 서버에 업로드)한다.(push)

5. git 서버에 변경된 정보를 재 다운로드 받는다. (pull)
